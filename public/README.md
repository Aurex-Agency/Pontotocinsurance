# Public Assets

This folder contains static assets for the Pontotoc Insurance Agency website.

## Logo Files
Place your logo files here with these recommended names:
- `logo.png` - Main logo (PNG format)
- `logo.svg` - Vector logo (SVG format, recommended)
- `logo-light.png` - Light version for dark backgrounds
- `favicon.ico` - Website favicon

## Usage
Files in this folder can be referenced in your components using `/filename.ext`

Example:
```jsx
<img src="/logo.svg" alt="Pontotoc Insurance Agency" />
```

## File Formats
- **SVG** - Best for logos (scalable, small file size)
- **PNG** - Good for logos with transparency
- **JPG** - For photos (not recommended for logos)
- **ICO** - For favicons
